# LL3D

A C++/DirectX11 3D game engine.

## Status

**Under development.**

## Features

- Unity-like Scene-GameObject-Component System
- Graphics
  - Camera
  - Light (Ambient, Directional, Point and Spot)
  - Load and Display model and material from mutiple type of files.
  - Fog
  - Mirror
  - Shadow
  - ...
- Input
- Math
